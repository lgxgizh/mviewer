# 图像查看与分析软件 — 需求规格说明书

## 项目背景

程序员日常工作中需要快速对比多张图片（如前后效果图、算法输出对比等），并测量特定区域的亮度和 RGB 通道统计数据。现有工具要么快但不支持框选统计（如 FastStone），要么支持框选统计但太慢（公司内部工具）。本项目旨在开发一款**既快又支持框选统计**的看图分析工具。

---

## 1. 项目概述

开发一款 Windows 平台下的图像查看与分析软件，核心参考 FastStone Image Viewer。支持高效浏览目录中的图片、多图并排比较、亮度直方图显示、以及框选区域统计功能。

**技术栈**：C++ / Qt 6 Widgets

---

## 2. 界面布局

整体布局分为三个区域：

```
┌──────────┬───────────────────────────────────┐
│          │                                   │
│  目录树  │         主图片视窗                │
│          │                                   │
│          │                                   │
├──────────┤                                   │
│ 缩略图   │                                   │
│ 列表     │                                   │
└──────────┴───────────────────────────────────┘
```

- **左侧上部**：目录树，浏览文件系统
- **左侧下部**：缩略图列表，显示当前目录中的图片
- **右侧**：主图片视窗，显示当前选中图片或比较视图

---

## 3. 核心功能

### 3.1 目录浏览

- 目录树支持展开/折叠
- 点击目录后，缩略图列表显示该目录下的所有图片
- 目录规模：单目录 ≤ 1000 张图片
- 排序方式：默认按文件名排序，支持按修改日期、文件大小、分辨率排序（点击列头切换）

### 3.2 缩略图列表

- 仅显示当前目录中的所有图片（非选中某个才显示，而是切换目录就刷新）
- 按目录中图片顺序排列
- 点击缩略图时，主视窗显示对应图片

### 3.3 图片查看

- 支持格式：JPEG、BMP、PNG
- 不支持 RAW、GIF
- 默认显示：适应窗口大小
- 缩放：鼠标滚轮
- 拖动：鼠标左键按住拖动
- 响应时间：点击后 < 100ms 显示完整视图

### 3.4 多图比较

#### 进入方式
- 在缩略图列表中选中 2~8 张图片
- 点击"比较"按钮进入比较模式

#### 布局规则（用户不可更改）

| 图片数量 | 布局 |
|---------|------|
| 2 张 | 左右并排 |
| 3 张 | 左中右并排 |
| 4 张 | 2×2 网格 |
| 5~8 张 | 2×4 网格 |

#### 同步控制
UI 上有一个勾选框，控制同步状态：

- **缩放同步**：
  - 勾选时：所有比较图片同步缩放
  - 未勾选时：仅鼠标当前所在图片缩放
- **拖动同步**：
  - 勾选时：所有比较图片同步移动
  - 未勾选时：仅鼠标当前所在图片移动

### 3.5 直方图

- 仅显示亮度直方图
- 半透明叠加显示在图片角落
- 比较模式下，所有图片均显示直方图

### 3.6 框选区域统计

- 框选形状：矩形
- 统计量：亮度均值、RGB 均值（R均值、G均值、B均值）
- 展示方式：在 UI 固定角落显示统计结果面板
- 比较模式下：对所有比较图片的同一区域进行统计并显示

## 4. 性能要求

### 4.1 图片尺寸
- 常见尺寸：4096 × 3072
- 最大尺寸：16384 × 12288（4倍于常见尺寸）

### 4.2 预加载
- 当前图片的前后相邻图片需要预加载
- 保证切换图片时快速显示

### 4.3 内存缓存
- 内存中缓存约 5 张已解码的图片
- 采用 LRU 策略管理

### 4.4 磁盘缓存
- 缩略图持久化到磁盘
- 下次打开同一目录时秒开
