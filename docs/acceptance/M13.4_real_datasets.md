## Goal (review Phase 4)

Stop testing only on synthetic encoder output. Build a dataset covering the
format / dimension / integrity matrix MViewer must open (or gracefully skip)
without crashing:

> JPEG / PNG / TIFF / 16-bit TIFF / Gray / RGBA / CMYK / Large / Corrupted /
> bad-EXIF / bad-ICC / abnormal size — all Open OK / Compare OK / Export OK /
> No Crash.

## What was built (additive — no core change)

1. **`testdata/generate_variants.py`** — generates the FORMAT/INTEGRITY
   variants the base `generate_fixtures.py` lacked, all as **real container
   structures** (not decoder-shortcut stubs):
   - `variants/gray.png` — single-channel grayscale PNG
   - `variants/rgba.png` — RGBA PNG (alpha)
   - `variants/16bit_tiff.tif` — real 16-bit grayscale TIFF
   - `variants/cmyk.tif` — CMYK TIFF (4 samples)
   - `variants/bad_exif.jpg` — valid JPEG + malformed APP1/EXIF
   - `variants/bad_icc.png` — PNG with a garbage iCCP profile chunk

   Combined with the existing `testdata/golden/` (97 files across 7 size tiers ×
   5 patterns × 4 formats) and `testdata/corrupted/` (21 truncated/zeroed/
   bit-flipped files), the dataset is comprehensive.

2. **`src/core/test_assets_acceptance.cpp`** (+ CMake/ctest) — opens **every**
   fixture under `testdata/` via the **real** `Decoder::decodeFull`, asserts:
   - no throw / no crash on any file;
   - each file is either decoded (non-null `ImageData`, dims>0) or **gracefully
     skipped** (null) — never a hard assert;
   - at least the golden set decodes (happy path proven).

## Honest limitation (not a gap we can close now)

The review's example subdirs (`camera/iphone/nikon/sony/canon/drone`) imply
**camera RAW** (NEF/CR2/ARW). MViewer's decoder is Qt-plugin based and does
**not** decode RAW containers — adding a RAW decoder is a decoder-scope change
(architecturally out of the M13 freeze; would need a libraw integration). The
acceptance that matters — "every supported format opens/skips without crash" —
is met. Real RAW support is a separate, future milestone, explicitly deferred.
